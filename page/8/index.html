Running ASP.NET Applications On Linux Using Mono

ASP.NET is a Windows based web technology. However it’s no more true, now we can run ASP.NET applications on a Linux environment using Mono Framework. Mono has developed a stand alone web server called Mono XSP for running ASP.NET applications on C# which can be run using Mono. Moreover it can be run on Apache web server using mod-mono Apache module. At the moment Mono XSP2 supports ASP.NET 1.0 and 2.0 versions. »

Mail Forwarding Can Make Spam

Forwarding a mail to other people can be a valuable thing to make others informed on a given subject. However adding your entire friends email addresses on the ‘To’ section can do a great harm even you without knowing. When people forward mails over and over again as mentioned, at a given time a mail can contain hundreds of email addresses. So when this kind of a mail reaches a person or a company who is looking for email addresses to be used for their advertising and spamming activities they extract the addresses and update their databases. »
